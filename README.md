| File name | Used for | Suggested size |
|---|---|---|
| `hero-band.jpg` + `hero-band.webp` | Header background photo | wide, landscape, 1600px+ width |

| `campus-splash.jpg` + `campus-splash.webp` | CampusGuide phone screenshot 1 | 455 × 1056 |
| `campus-chat-removebg-preview.png` + `.webp` | CampusGuide phone screenshot 2 (transparent background) | 406 × 614 |
| `vr-product-removebg-preview.png` + `.webp` | VR Glow phone screenshot 1 (transparent background) | 368 × 679 |
| `vr-catalog-removebg-preview.png` + `.webp` | VR Glow phone screenshot 2 (transparent background) | 374 × 666 |

Notes:
- Files ending in `-removebg-preview` are meant to be transparent PNGs (phone mockups with the background removed), so they float without a card behind them.
- `.webp` versions are optional but recommended (used first when the browser supports it, `.jpg`/`.png` is the fallback).
- Delete this `README.md` once the real images are in place — it won't affect the site either way.
